# Flipnote Hatena server (Nintendo DSi)
> This is a replacement server of the now discontinued "Flipnote Hatena service" from 2008 - 2013
# Nintendo DSi Headers
here are the list of Nintendo DSi headers that get used while connecting or in use of Flipnote Hatena:

X-DSi-MAC
> The Nintendo DSi MAC Adress seen in the Internet Settings (Uppercase HEX)
 
X-DSi-ID

> A User's Flipnote Studio ID (uppercase HEX, 16 Digits)
 
X-DSi-Auth-Response
> 8-byte lowercase HEX string, based on auth challenge value
 
X-DSi-SID
> Hatena's Session ID (shut Down 2018)
 
X-Ugomemo-Version
> The Flipnote Studio app version (0, 1, or 2)

X-DSi-User-Name
> The Username of the DSi console
 
X-DSi-Region
>The DSi Console region
 
X-DSi-Lang
> 2 letter regions (us, eu or jp)
 
X-DSi-Country
> 2 Letter Country code (Uppercase)
 
X-Birthday
> A User's DSi Birthday (YYYYMMDD Format)

X-DSi-DateTime
> 	DSi Console date and time (format is YYYY-MM-DD_hh:mm:ss)
 
X-DSi-Color
> 	DSi User's favorite color (lowercase HEX string, rgb555 format)
